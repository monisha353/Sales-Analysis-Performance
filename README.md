# Sales Performance Analysis using SQL

## Project Overview
This project analyzes sales data using SQL to derive insights into revenue, profit, customer behavior, and product performance. The dataset is structured in a relational format to simulate a real-world business scenario.

## Objectives
- Calculate total revenue and total profit
- Identify top-selling products and categories
- Analyze customer-wise spending
- Track sales trends
- Practice joins and aggregations

## Dataset
The project uses four tables:
- customers
- products
- orders
- order_details

## Tools Used
- SQL (MySQL)

## Key SQL Concepts
- JOINs
- GROUP BY
- Aggregate Functions
- Foreign Keys
